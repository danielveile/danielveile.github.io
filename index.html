HTML <head>

Header Tags

Tag Name Description
<title> Title Defines the title that shows in the tab.
<base> Base This HTML element specifies the base URL to use for all relative URLs in a document. There can be only one <base> element in a document.
<link> Link Use <href> to link the HTML document to another document, specifying what that document does to the HTML. Usually used to link CSS stylesheet.
<style> Style Use this to style the HTML document. This element contain CSS code. Not advicable.
<meta> Meta Provide extra meta data.
<script> Script Use this to add JS scripts within the HTML document.
<noscript> No Script Element to display in the webpage if scripting is disabled in the browser.
<template> Template An element where we define a template for how a certain HTML element should look like. Very good to use with JS scripting.
